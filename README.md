# STM32G431CBU6 Sandbox

This repository is a small test and development environment for the STM32G431CBU6 dev board.

It is intended for experimenting with board bring-up, peripherals, firmware changes, and other quick validation work on real hardware.

The project is generated around STM32CubeMX/CMake and kept intentionally lightweight so it can be used as a practical sandbox rather than a production application.
